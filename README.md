# Retail Sales Analytics Hackathon Project

## 📌 Overview
Retail Sales Analytics is a data analysis project developed to analyze retail transaction data and identify meaningful business insights. The project focuses on sales performance, customer behavior, product analysis, and revenue trends using Python and data visualization techniques.

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🎯 Objective

The main objective of this project is to analyze retail sales data and generate business insights related to:
- Sales trends
- Customer purchasing behavior
- Product performance
- Revenue growth

---

## 📊 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Sales Trend Analysis
- Customer Analysis
- Product Category Analysis
- Data Visualization

---

## 📈 Key Insights

- Electronics category generated the highest revenue
- Weekend sales were higher compared to weekdays
- Repeat customers contributed more revenue
- Seasonal sales growth observed during festive periods
